1 パッケージ構成

ディレクトリ構造 必須
src/main/java以下のパッケージ構造
src/main/java/
└── com.example.taskapp/
    ├── TaskAppApplication.java        # メインクラス
    ├── config/
    │   ├── SecurityConfig.java        # Spring Security設定
    │   └── WebConfig.java             # Web設定
    ├── controller/
    │   ├── TaskController.java        # タスク管理
    │   ├── UserController.java        # ユーザー管理
    │   └── LoginController.java       # 認証
    ├── service/
    │   ├── TaskService.java           # タスクビジネスロジック
    │   └── UserService.java           # ユーザービジネスロジック
    ├── repository/
    │   ├── TaskMapper.java            # タスクDBアクセス
    │   └── UserMapper.java            # ユーザーDBアクセス
    ├── entity/
    │   ├── Task.java                  # タスクエンティティ
    │   ├── User.java                  # ユーザーエンティティ
    │   └── Category.java              # カテゴリエンティティ
    └── form/
        ├── TaskForm.java              # タスク入力フォーム
        └── LoginForm.java             # ログインフォーム

（上記を参考に編集してください）
パッケージの役割 必須
各パッケージの責務を説明
パッケージ 役割 使用するアノテーション
config アプリケーション設定 @Configuration, @EnableWebSecurity
controller リクエスト/レスポンス処理 @Controller, @GetMapping, @PostMapping
service ビジネスロジック実装 @Service, @Transactional
repository データベースアクセス @Mapper
entity データモデル定義 @Data, @NoArgsConstructor
form 画面入力データ受け取り @Data, @NotBlank, @Size

3 Repository（リポジトリ）

TaskMapper.java 必須
タスクリポジトリのインターフェース定義
┌────────────────────────────────────────────────┐
│              《Interface》                      │
│               TaskMapper                        │
│              @Mapper                            │
├────────────────────────────────────────────────┤
│ + selectAll(): List<Task>                     │
│ + selectById(id: Long): Task                   │
│ + selectByUserId(userId: Long): List<Task>    │
│ + insert(task: Task): void                     │
│ + update(task: Task): void                     │
│ + delete(id: Long): void                       │
│ + search(keyword: String): List<Task>         │
└────────────────────────────────────────────────┘

（上記を参考に編集してください）
メソッド 引数 戻り値 SQL種別 説明
selectAll なし List<Task> SELECT 全件取得
selectById id: Long Task SELECT ID指定取得
selectByUserId userId: Long List<Task> SELECT ユーザー別取得
insert task: Task void INSERT 新規登録
update task: Task void UPDATE 更新
delete id: Long void DELETE 削除
